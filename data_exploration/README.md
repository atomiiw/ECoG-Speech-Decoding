# Data Exploration

Notebook for exploring and preparing the Cogan Lab ECoG dataset.

## What's here

- `explore_pkl.ipynb`: structure inspection, channel and trial layout, basic statistics, format conversion utilities for the Cogan Lab `.pkl` data

## Dataset

Cogan Lab (Duke) micro ECoG recordings during production of nonsense syllables.
8 patients, 144 trials per patient, 3 phonemes per trial, 9 possible phonemes, 200 time points (±500 ms around phoneme onset).

The actual `.pkl` data files are not included for IRB reasons.
